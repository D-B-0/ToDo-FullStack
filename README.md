# ToDo-FullStack
A simple ToDo fullstack application

To get started execute the following comand while in the main folder
&nbsp;
`git submodule update --init --recursive`
&nbsp;
Then in the server folder
&nbsp;
`
npm i
&nbsp;
npm run start
`
&nbsp;
And last in the client folder
&nbsp;
`
npm i
&nbsp;
npm run serve
`
